# 黄氏家族寻根平台 - API连接问题解决方案

## 问题描述

您遇到的 "无法加载模型列表，请检查网络连接" 错误是由于前端无法连接到后端API服务所致。这通常表现为 "Unexpected token '<'" 错误。

## 问题原因

1. **前端已部署**: 前端文件已成功部署到FTP服务器 (qxu1606470020.my3w.com)
2. **后端未部署**: 后端API服务未部署到同一服务器
3. **API连接失败**: 前端尝试连接API时无法找到后端服务

## 解决方案

### 方案一：部署后端API服务

如果您有服务器可以运行Node.js后端服务：

1. **部署后端服务**:
   ```bash
   # 在服务器上克隆项目
   git clone <repository-url>
   cd hxfund
   
   # 安装依赖
   npm install
   
   # 配置环境变量
   cp server/config/.env.example server/config/.env
   # 编辑 .env 文件，配置正确的设置
   
   # 配置API密钥
   node qwen-code.js --init
   
   # 启动服务
   npm start
   ```

2. **更新前端API配置**:
   修改 index.html 中的 API_CONFIG 指向您的后端服务地址

### 方案二：使用已有的后端服务

如果您已有运行中的后端服务，只需更新前端配置：

1. 编辑部署后的 index.html 文件
2. 修改 window.API_CONFIG.baseURL 为您的后端服务地址

### 方案三：本地测试

如果要在本地测试完整功能：

1. 在本地启动后端服务:
   ```bash
   cd hxfund
   npm start
   ```

2. 访问 http://localhost:3000
